Personal Task Manager Application

A personal execution dashboard designed to track deep work and daily habits.

Features:
- Execution grid for weekly discipline
- Automatic weekly scoring
- Deep work tracking (DSA / DevOps / Projects / PSU's)
- Daily completion rings
- Performance dashboard with charts
- Local storage persistence

Built with:
React + Vite + Recharts
